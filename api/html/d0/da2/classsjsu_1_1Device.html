Detailed Description

template<class DeviceProtocol, Endian endianess, template< Endian endian, WriteFnt write, ReadFnt read > class MemoryMap>
class sjsu::Device< DeviceProtocol, endianess, MemoryMap >

Device takes a MemoryMap structure and a Device communication protocol and combines them to create a hardware abstraction that allows the remote device to be accessed as if it were apart of the local memory map.

Template Parameters
DeviceProtocol- Device communicate protocol
endianess- the endianess of the device to communicate with
MemoryMap- the memory map structure to use.
