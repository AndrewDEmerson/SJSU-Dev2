Detailed Description

Abstract interface for a platform's System Controller. A System controller manages a platforms:

  • Clocks and their speeds
  • PLLs feeding into the system's clocks
  • Manages power systems of peripherals and system blocks

The System controller also gives information about the system such as, what speed is a peripheral running at so that their peripheral drivers can calculate things such as the clock rates for serial communicate or the clock cycles before an event or interrupt is planned to occur.
