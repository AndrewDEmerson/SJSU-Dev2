virtual void sjsu::rtos::TaskInterface::SetDelayTime ( uint32_t  time)
pure virtual

Sets the delay time to ensure Run() is called at a desired frequency for periodic tasks.

For examples, if time = 1000 and the sjsu::rtos tick period = 1ms, then Run() will be called every 1 second.

Parameters
timeDesired delay time in sjsu::rtos ticks.
