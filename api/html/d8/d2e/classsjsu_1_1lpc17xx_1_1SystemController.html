Sets the divider to control the desired peripheral clock rate (PCLK) for a specified peripheral where PCLK = CCLK / divider.

The following dividers are supported for non-CAN peripherals: 1, 2, 4, 8. For CAN the following dividers are supported: 1, 2, 4, 6.

Parameters
peripheral_selectPeripheral to configure.
peripheral_dividerPeripheral clock divider value.

◆ SetSystemClockFrequency()

void sjsu::lpc17xx::SystemController::SetSystemClockFrequency ( units::frequency::megahertz_t  frequency) const
inlineoverridevirtual

Sets a desired CPU speed by using the internal RC as the oscillator source and configuring PLL0.

Note
The USB subsystem should be configured to obtain its input clock from the USB PLL. The internal RC is used for PLL0 and will not generate a precise enough clock to be used by the USB subsystem.
Parameters
frequencyThe desired CPU Clock frequency in megahertz.
