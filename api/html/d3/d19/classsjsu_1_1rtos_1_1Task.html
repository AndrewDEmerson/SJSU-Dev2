Checks if the object was statically allocated either in the .data, .bss, or on the heap. Returns false if the position of this object is not within the bounds of those sections meaning it must be on the heap, which means that the object may not live for the total lifetime of program. Sets the delay time to ensure Run() is called at a desired frequency for periodic tasks.

For examples, if time = 1000 and the sjsu::rtos tick period = 1ms, then Run() will be called every 1 second.
