Initialize and enable hardware. This must be called before any other method in this interface is called.

Parameters
counter_frequency- the frequency that the timer's count register will increment by. If this is set to 1'000'000Hz then the counter will increment every microsecond. register will be 10 ms.
callback- a callback that will be called when the condition set by SetTimer method has occurred.
priority- sets the Timer interrupt's priority level, defaults to -1 which uses the platforms default priority.

Set a timer to execute your timer command when the time counter equals the match register. time in ticks dependent on initialization Functionality is defined by mode: interrupt, stop, or reset on match.

Parameters
match_count- when the time counter matches this number perform the action in the "action" parameter.
action- the action to take when timer counter equals the match register.
match_register- which match register should be used for holding the count for the action.
