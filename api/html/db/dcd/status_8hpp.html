} \

})

A macro for simplifying the boiler plate of evaluating an expression that returns a Returns<> object. This will handle evaluating the function that returns a Returns<> object and if it is an error, it will return/bubble up the error code rather than continuing through the code. Otherwise, this macro will return the value of the result.

Usage examples can be shown at the top of the file.
