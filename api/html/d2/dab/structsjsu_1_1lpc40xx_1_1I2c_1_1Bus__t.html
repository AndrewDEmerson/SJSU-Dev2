Detailed Description

Bus_t completes the partial bus structure by including a ISR handler. The ISR handler is intended to point to an instance of the I2cHandler template function. The template parameter would be a const reference to a statically allocated PartialBus_t. That way, The actual handler (non-template) that requires a reference to a PartialBus_t can have it passed to it via a template parameter.
