void sjsu::rtos::TaskScheduler::AddTask ( TaskInterface *  task)
inline

Add a task to the task scheduler. If the scheduler is full, the task will not be added and a fatal error will be asserted.

Note
When a task inheriting the TaskInterface is constructed, it will automatically call this function to add itself to the scheduler.
