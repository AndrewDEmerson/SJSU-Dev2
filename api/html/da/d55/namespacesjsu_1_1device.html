endianess- is the register big or little endian

Represents a contiguous block of memory in the memory map.

Template Parameters

kLength- number of bytes in the array

Can swap from big to little indian.

Handler for value assignment of the register. This will attempt to write use the write function to write to the register in the defined memory map.

Returns
true - if the assignment was successful
false - if the assignment was a failure
Int - when this register is accessed like an integer, this function will fetch and return the latest value of that register from the memory map.
